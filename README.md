# cloud-lab-melissa-ameryahia
**Cloud Computing Lab – GitHub Codespaces & Apache
**
This repository demonstrates the basics of cloud-based web development using GitHub Codespaces and the Apache web server. setting up a cloud environment, hosting a custom web page, and understanding core concepts of cloud computing applied to web development.

**Features & Learning Goals
**
- Create and use a cloud-based development environment with GitHub Codespaces.
- Install and configure an Apache web server in the cloud.
- Edit and serve a custom web page.
- Forward a port to access the website from a browser.
- Understand the basics of cloud computing applied to web development.
